================================================================================
                    Nano Banana - Gemini 이미지 생성기
                    dothome 웹호스팅 설치 가이드
================================================================================

[ 파일 구성 ]
- index.html    : 메인 웹 인터페이스
- api.php       : Gemini API 프록시 서버
- config.php    : 설정 파일 (API 키 입력 필요)
- .htaccess     : Apache 서버 설정 및 보안

================================================================================

[ 설치 방법 ]

1. Gemini API 키 발급
   - https://aistudio.google.com/app/apikey 접속
   - Google 계정으로 로그인
   - "Create API Key" 클릭하여 키 발급
   - 발급받은 키 복사

2. config.php 수정
   - config.php 파일을 텍스트 에디터로 열기
   - 'YOUR_API_KEY_HERE' 부분을 발급받은 API 키로 교체

   예시:
   define('GEMINI_API_KEY', 'AIzaSyC...');

3. dothome에 파일 업로드
   - dothome FTP 또는 파일 관리자 접속
   - 모든 파일을 public_html 또는 www 폴더에 업로드
   - sessions 폴더가 자동으로 생성됩니다

4. 접속 테스트
   - 브라우저에서 http://yourdomain.dothome.co.kr 접속
   - 프롬프트 입력 후 이미지 생성 테스트

================================================================================

[ 기능 안내 ]

1. 텍스트로 이미지 생성
   - 프롬프트를 입력하고 전송 버튼 클릭
   - 한국어/영어 모두 지원

2. 이미지 편집
   - 참조 이미지를 업로드하고 수정 요청
   - 예: "이 고양이에 마법사 모자를 씌워줘"

3. 멀티턴 대화
   - 생성된 이미지를 대화형으로 계속 수정 가능
   - "좀 더 밝게 해줘", "배경을 바꿔줘" 등

4. 모델 선택
   - Gemini 2.5 Flash: 빠른 생성 (1024px)
   - Gemini 3 Pro: 고품질, 최대 4K 해상도

5. 가로세로 비율
   - 1:1, 16:9, 9:16, 4:3, 3:4 등 다양한 비율 지원

================================================================================

이미지 편집 예시:
- "이 사진의 배경을 해변으로 바꿔줘"
- "이 로고를 파란색으로 변경해줘"
- "이 인물의 옷을 빨간 드레스로 바꿔줘"

[ 문제 해결 ]

Q: API 오류가 발생해요
A: config.php의 API 키가 올바른지 확인하세요

Q: 이미지가 생성되지 않아요
A: 프롬프트가 콘텐츠 정책을 위반하지 않는지 확인하세요

Q: 세션이 저장되지 않아요
A: sessions 폴더의 쓰기 권한(755 또는 775)을 확인하세요

Q: 업로드가 안 돼요
A: php.ini의 upload_max_filesize를 확인하세요 (최소 10M 권장)

- API 키를 공개적으로 노출하지 마세요
